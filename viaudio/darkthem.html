Alternar el mode fosc / clar

Aquesta pàgina selecciona automàticament el mode segons la preferència del teu navegador, però pots alternar el mode també amb el botó.

Feu clic al botó per alternar entre el mode fosc (Dark Mode) i el clar (Light Mode) per a aquesta pàgina.

Recordeu que amb JS podeu alternar entre diferents clases amb la següent comanda: document.body.classList.toggle("claseA");
